回测参数 (成本模型)

成本 = 悲观撮合假设; 上限 = 单批保护值(排名取数跟它); OOS = 训练段选/留出段一票否决, 比例长期固定 7:3

交叉测试门槛(批量回测勾"跨品种"时生效)

主货币成绩过线才展开交叉品种(省算力);没过/没测过的只跑主品种,下一批自动够格。 按 ID 点名回测不走门槛(点名即信任,随便交叉)。全留空 = 不设门槛(全交叉)

关2 对账参数

建议 2。正常成交滞后 <1 分钟(实测 3~8 秒), 2 已留足余量; 设置过大容易把相邻的两笔不同交易误配成同一笔(误判匹配), 经验法则:容差 < 最小K线周期的一半(M5→勿超2)

策略运行状态(strategy_runtime 区间记录)

worker 心跳持续记录每个策略的真实运行区间 → 关2对账只比“在跑的时段”。 写入间隔=运行状态多久落一次库(默认5); 裂段阈值=断多久算一次下架(默认15, 需>写入间隔)
