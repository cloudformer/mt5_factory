## 维度二:实例级(策略+参数 深度分析)—— 这一个实例为什么赢/为什么输

**必须两条腿**:基于回测数据库(期1)+ 实际交易分析 demo/live(期2,依赖 v0.2),对照才算完整归因:
"是因为拟合历史数据,还是短期行情,还是真规律"。

### 期1:基于回测数据库(backtests.trades 逐笔明细已在库)— ✅ 全部现在可做

| 分析 | 用什么 | 回答什么 |
|------|--------|---------|
| 盈利时间分布 | entry_time × points | 赚的钱**分散**(稳)还是**集中在几天**(靠单一事件,脆弱=拟合的另一种长相) |
| 时段/星期效应 | entry_time 按小时/星期聚合 | 只在某时段赚?(亚洲盘 vs 欧美盘) |
| 方向不对称 | dir × points | 多单赚空单亏 = 只是骑了单边行情,非策略有效 |
| 出场原因构成 | reason(tp/sl/**gap**) | gap 占比高 = 跳空风险大,实盘更疼 |
| 连亏串 / 权益曲线形态 | points 按时间累加 | 最长连亏、回撤期时长 — 实盘拿不拿得住 |
| 拟合判别·时间轴 | metrics.oos(已实现) | 训练 vs 留出 |
| 拟合判别·品种轴 | 跨品种行(已实现) | 本币 vs 他币 |

### 期2:实际交易分析(demo/live 胜负原因)— ⏳ 依赖 v0.2 trades 表(逐笔落库)

| 分析 | 回答什么 |
|------|---------|
| 回测预期 vs demo 实际**逐笔对账** | 回测模型对这个策略**准不准**(准入漏斗关2的本体) |
| 实盘胜负归因(期1同套指标跑在真实成交上) | 实盘的赢法和回测**是不是同一种赢法** |
| 偏离度量化(成交价/滑点/漏单) | 执行损耗多大 |
| 手动干预剔除(reason=manual 不算策略头上) | 战绩纯净度 |
| live vs demo 对照 | 真金环境是否复现 demo |
