## A. 指标搭配控制(生成页就地配置)【可开工】

门控模板(intraday_multi)已把"搭配"变成参数(`use_trend`/`use_atr` 就是指标开关) → **控制搭配 = 锁参数**,生成页参数空间表改为可编辑,每个参数三种玩法:

```
use_trend   [锁定 = 1]        ← 锁开关 = 定指标搭配
rsi_period  [范围 5 ~ 14]     ← 收窄采样范围
sl_atr      [默认 1.0~4.0]    ← 不动 = 按模板默认空间
```

- 生成请求带参数覆盖 → `random_combo` 在**用户定义的子空间**里采样;
- 改动面:api generate 接受 overrides + 生成页表格可编辑,模板代码零改动。

## B. 规模化生成 → 收敛漏斗(每批 2000 量级)【A 完成后即可手动跑】

2000 × 9 品种全跨品种 = 1.8 万次回测,傻跑不行 → **漏斗省算力**:

```
2000 参数(随机/AI) → 主品种回测(约30~60分钟)
  → 粗筛(config 阈值: 笔数≥N 且 净点>0, 预计剩 10~20%)
  → 幸存者跨品种回测(数百×9, 可承受)
  → OOS留出盈利 + 健壮≥60 双筛 → 真候选(预计个位~十位数)
```

- **v1(手动三步)**:现有功能已支撑 — 生成(数量2000) → 回测"仅未测试" → 幸存者按ID勾跨品种;单批上限已可配。
- **v2(一键管线)**:生成完自动触发回测 → 完成后自动对幸存者跑跨品种 → 出双筛榜单。
- **收敛度量**:每批记录漏斗数字(2000 → 粗筛剩X → 双筛剩Y);**Y/N = 该搭配方向的"含真率"**,批间对比即知往哪收敛(数据喂 v1.4 维度一)。
- 回归测试:同种子+同数据窗口可复现一批(生成已带随机种子机制的按需补)。
